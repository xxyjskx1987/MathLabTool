## 特点

- **无依赖性：** 消除固定的中心化节点依赖，消除平台化应用垄断。  
- **自组织：** 不依赖固定的中心化节点，完成P2P网络自动组建。   
- **自治：** 可以自动完成网络节点的修复，保证通信顺畅。  
- **可监管：** 无论何时何地，应用可以被指定人员监管。  
- **可追溯：** 数据来源可以被追溯，数据归属可以被证明。  
- **开放性：** 所有服务都可以被参与，所有服务都可以被选择，拥有全部的数据主权。
